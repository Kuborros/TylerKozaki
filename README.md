Mod commision made by [Kubo](https://github.com/Kuborros), for [Tyler Yasha](https://bsky.app/profile/tyleryasha.bsky.social).

It includes Tyler's character _Tyler Kozaki_ as a playable character in Freedom Planet 2.

### Features:

- Custom character art and sprites!
- Custom voices!
- Two custom music tracks!
- 7 extra badges!

Released on Gamebanana: **https://gamebanana.com/mods/651473**
